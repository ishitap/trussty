Help

n node mode — add nodes and move them around

t truss mode — add trusses by dragging between nodes

f force mode — add forces by dragging them from nodes

g fixed joint — select a fixed joint node by clicking on it

r rolling joint — select a rolling joint node by clicking on it

s toggle snap to grid — useful for nodes in straight lines and forces that go straight down

shift + click remove an element
